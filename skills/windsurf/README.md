# Windsurf Rules

Windsurf-specific rules derived from universal-dev-standards.

## Status

✅ **Complete** - Ready to use

## Installation

Copy the `.windsurfrules` file to your project root:

```bash
cp skills/windsurf/.windsurfrules .windsurfrules
```

Or copy from this repository directly:

```bash
curl -o .windsurfrules https://raw.githubusercontent.com/AsiaOstrich/universal-dev-standards/main/skills/windsurf/.windsurfrules
```

## What's Included

The `.windsurfrules` file includes guidance for:

- **Commit Messages** - Conventional Commits format
- **Code Quality** - Best practices and principles
- **Security** - Common vulnerabilities to avoid
- **Testing** - Testing pyramid and patterns
- **Git Workflow** - Branch naming conventions
- **Documentation** - File structure standards
- **AI Collaboration** - Guidelines for AI interaction
