# 回顾助手

> **语言**: [English](../../../../skills/retrospective-assistant/SKILL.md) | 简体中文

引导结构化的团队回顾，识别改善机会并追踪行动项目。

## 指令

| 指令 | 说明 |
|------|------|
| `/retrospective` | 交互式 Sprint 回顾 |
| `/retrospective release` | Release 回顾（含指标） |
| `/retrospective actions` | 列出未完成行动项目 |
