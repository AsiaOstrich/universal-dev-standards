## 安装

将 `.clinerules` 档案复制到您的专案根目录：

```bash
cp skills/cline/.clinerules .clinerules
```

或直接从此仓库复制：

```bash
curl -o .clinerules https://raw.githubusercontent.com/AsiaOstrich/universal-dev-standards/main/skills/cline/.clinerules
```
