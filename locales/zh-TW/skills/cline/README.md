# Cline 規則

衍生自 universal-dev-standards 的 Cline 專用規則。

## 狀態

✅ **完成** - 可直接使用

## 安裝

將 `.clinerules` 檔案複製到您的專案根目錄：

```bash
cp skills/cline/.clinerules .clinerules
```

或直接從此倉庫複製：

```bash
curl -o .clinerules https://raw.githubusercontent.com/AsiaOstrich/universal-dev-standards/main/skills/cline/.clinerules
```

## 包含內容

`.clinerules` 檔案包含以下指引：

- **提交標準** - Conventional Commits 格式
- **程式碼審查** - 品質檢查表
- **測試** - 測試金字塔和 FIRST 原則
- **Git 工作流程** - 分支命名
- **文件** - 檔案慣例
- **AI 最佳實踐** - AI 協作指南
