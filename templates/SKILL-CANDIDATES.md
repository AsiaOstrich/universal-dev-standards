# Skill Candidates | Skill 候選清單

> **The 3-Times Rule | 3 次法則**
>
> Same process done manually ≥ 3 times → record here → at 3 → run `/process-to-skill`
> 同一流程手動執行 ≥ 3 次 → 記錄於此 → 達 3 次 → 執行 `/process-to-skill`

## How to Use | 使用方式

1. When you notice a repeated process, add a row to Candidates below
   發現重複流程時，在「候選清單」表格新增一行
2. Each time you repeat it manually, increment the Count
   每次重複手動執行時，更新「執行次數」
3. When Count reaches 3, mark Trigger ✅ and run `/process-to-skill`
   次數達 3 時，觸發欄填 ✅，執行 `/process-to-skill`
